dom html 문서에 있는 내용을 자바스크립트로 변경, 추가,삭제,변경 등등을 하기위한 모델
html 요소에 접근을 해야함. 태그,id, class... 등등의 선태갖로 해당하는 엘리먼트를 가져온다

요소를 선택하는 방법

id로 선택
document.getElementById('아이디'); 메서드를 활용
class로 선택
document.getElementsByClassName('클래스');
tag로 선택
document.getElementsByTag('태그명');
선택자로 선택
document.querySelector('선택자'); document.querySelectorAll('선택자');
